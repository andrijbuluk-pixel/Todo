# Todo list


### This project is a user-friendly task management website. It allows you to create, delete, and edit tasks, as well as flexibly manage their tags.

Key features:

- Clear status tracking: you can mark a task as completed with a single click.
- Deadline monitoring: the application allows you to track the remaining time to complete a task, ensuring you never miss a thing.
